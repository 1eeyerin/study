# 스코프

```javascript
function add(x, y) {
    console.log(x, y); // 3 6
    return x + y;
}

add(3, 6);

console.log(x, y);
// x is not defined
```

변수, 함수, 클래스와 같은 식별자가 본인이 선언된 위치에 따라 **다른 코드에서 참조될 수 있는지 없는지 결정하는** 유효 범위를 스코프라고 한다.

스코프는 변수의 가시성과 **생존기간을 관리**하며,\
코드내 **충돌을 방지**하고, 변수 **이름을 구분하는 역할**을 한다.

## 스코프 종류

선언 시점에 따라 전역에서 선언되었다면, **전역 스코프**를 가지고 지역에서 선언되었다면 **지역 스코프**를 가진다.

전역에 선언된 변수는 **전역 스코프**를 가져 어디에서든 참조가 가능하지만, 지역에서 선언된 **지역 스코프**는 지역에서만 접근할 수 있다.

## 스코프 레벨

### 함수 스코프

`var`

함수 내부에서 선언된 변수는 함수 외부에서 접근할 수 없다.

### 블럭 스코프

`let` `const`

블럭 내부에서 선언된 변수는 블럭 외부에서 접근할 수 없다.

## 스코프 체인

스코프가 변수를 검색할때 내부에서 외부로 계층적으로 검색하는 것을 말하는데, 내부 스코프에서 변수를 찾을 수 없다면, 외부 스코프에서 검색을 한다.

1. 정확한 변수나 함수를 참조할 수 있다.
2. 변수 이름 충돌을 막는다.
